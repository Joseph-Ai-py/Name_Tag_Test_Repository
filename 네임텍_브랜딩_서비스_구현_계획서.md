# PART 0. 기존 시스템 문제 진단 및 재설계 방향

---

## 0-1. 기존 시스템 문제 진단

현재 구현된 백엔드(`README.md`, `BACKEND_FLOW_EXPLANATION.ipynb` 기준)는 다음 구조다:

```
사용자 입력 (4개 필드) → AI 1회 호출 → 단일 결과 출력
```

**문제점:**

- 단방향 파이프라인: 사용자가 중간에 선택하거나 조합할 수 없음
- 결과물이 브랜드명 3개 + 타이포그래피 + 캐릭터 수준에 그침 (얕음)
- 고객에게 전달할 PDF 보고서 수준의 콘텐츠 깊이가 없음
- 세션 간 상태(state) 관리가 없어 다단계 선택 흐름 불가
- LLM 레퍼런스 PDF에서 요구하는 항목(컬러 시스템, 톤앤매너, 슬로건, 브랜드 스토리, 타이포그래피 스펙, 마케팅 전략 등)을 생성하는 로직이 없음

---

## 0-2. 신규 시스템 전체 플로우 (토스(Toss) 방식)

```
[Phase 1] 설문 수집 (5단계)
  └─ 업종, 감성 키워드, 타겟, 컬러 무드, 추가 키워드 입력

[Phase 2] 초안 후보 3개 생성 (AI 1차 호출)
  └─ 각 후보: 브랜드명 + 의미 + 슬로건 + 브랜드 스토리(요약) + 핵심 컬러 1개

[Phase 3] 사용자 선택 & 조합
  └─ 후보 1개 선택 OR 교차 선택 (예: 1번 브랜드명 + 2번 스토리)
  └─ 선택 결과를 서버 세션에 저장

[Phase 4] 확장 생성 (AI 2차 호출)
  └─ 선택된 브랜드 정체성 기반으로 전체 가이드 생성
  └─ 컬러 팔레트 5종, 타이포그래피, 톤앤매너, 마케팅 방향, 캐릭터 등

[Phase 5] PDF 가이드 실시간 생성
  └─ 화면 옆에 완성된 PDF 미리보기 렌더링
  └─ 다운로드 버튼 제공
```

---

## 0-3. 기존 vs 신규 비교

| 항목 | 기존 | 신규 |
|---|---|---|
| 플로우 | 단방향 1회 출력 | 5단계 대화형 선택 플로우 |
| AI 호출 횟수 | 1회 | 2회 (후보 생성 + 전체 가이드) |
| 결과물 깊이 | 브랜드명 + 타이포그래피 수준 | 9페이지 분량 브랜드 가이드북 |
| 사용자 선택권 | 없음 | 교차 선택 (토스 방식) |
| PDF 출력 | 없음 | WeasyPrint 동적 PDF 생성 |
| 세션 관리 | 없음 | 메모리 세션 (Redis 전환 가능) |
| 엔드포인트 수 | 2개 | 7개 이상 |
| 컬러 | HEX 1개 | 5색 팔레트 (HEX + 역할 + 설명) |
| 타이포그래피 | 폰트명만 | 폰트 + 스타일 + 웨이트 + 사용처 |
| 캐릭터 | 이미지 생성만 | 이름 + 특징 + 선택이유 + 스토리 |

## 1-1. Perplexity 보고서

**브랜드명 제안:** 온기상점, 하루의 결, 발견의 방, 오후의 소품, 포근한 장면
**슬로건:** 일상에 놓는 작은 온기 / 당신의 공간을 다정하게 / 매일의 취향을 발견하는 곳

| 섹션 | 핵심 내용 |
|------|-----------|
| 브랜드 코어 | 미션·비전·약속 구조화. "일상 속 작은 발견으로 공간의 온도를 높이는 큐레이션 브랜드"로 포지셔닝 |
| 고객 페르소나 | 30대 직장 여성, 홈카페·독서·향·조명 소품에 반응. 감정 트리거 4가지 제시(위로·자기표현·발견감·아날로그 감각) |
| 포지셔닝 | 일반 소품몰 vs 본 브랜드 5축 비교표(제안방식·분위기·구매이유·메시지·기준) |
| 브랜드 아이덴티티 | 키워드 구조(Emotional/Lifestyle/Discovery/Material) 4개 클러스터로 분류 |
| 톤앤매너 | 6개 축 스펙트럼 수치 제시(Formal↔Friendly 70%, Careful↔Spontaneous 65% 등) |
| 비주얼 전략 | 컬러 방향(크림·세이지그린·뮤트테라코타), 촬영 스타일(오전 햇살·커튼 빛), 디자인 요소(둥근모서리·종이질감) |
| 상품 전략 | 카테고리 4개(데스크/문구·리빙/패브릭·무드/휴식·수납/정리) + 시즌셀렉션 |
| 콘텐츠 전략 | 4개 기둥(공간제안형·발견형·루틴형·브랜드가치형) + 채널별 운영 방향 |
| 실행 전략 | 4단계 로드맵(브랜드세팅→콘텐츠세팅→판매운영→고객관계강화) |

**강점:** 톤앤매너 수치화, 차별화 포지셔닝 비교표, 상품 카테고리 구조화
**약점:** 컬러 헥스코드 미제시, 타이포그래피 미제시

## 1-3. Adobe Firefly 보고서

**브랜드명 제안:** 온기일상(Ongi Ilsang / Warmday)
**슬로건:** "당신의 일상에 온도를 더하다" / "Warm your everyday"

| 섹션 | 핵심 내용 |
|------|-----------|
| 브랜드 핵심가치 | 온기(Warmth)·발견(Discovery)·진정성(Authenticity) 3축 |
| 브랜드 퍼소나 | 32세 마포구 콘텐츠기획자, 퇴근 후 화분·허브차 루틴, 무인양품·핀란드디자인 선호 |
| 컬러 팔레트 | Primary #D4B896(웜베이지) / Secondary #7A8C6E(세이지그린) / Accent #C4775A(테라코타) / Neutral #F5F0EA / Text #3A3028 (5색 + 역할 명시) |
| 타이포그래피 | 헤드라인: 나눔명조/Cormorant Garamond / 바디: Pretendard/Inter / 포인트: 손글씨(고운밤체) |
| 포지셔닝 맵 | 고감성/저가 ~ 고감성/중가 좌표: "접근 가능한 감성(Accessible Sensibility)" |
| 채널 전략 | 인스타·핀터레스트·카카오채널·블로그/브런치·네이버스마트스토어 5채널 목적·전략 |
| 콘텐츠 기둥 | 발견의 순간·당신의 공간·온기 있는 하루 3기둥 |
| 고객 여정 | 인식→탐색→구매→재구매→팬화 5단계, 채널 매핑 포함 |
| 가격 전략 | 15,000~45,000원 앵커, 스타터세트 3만원대, 시즌컬렉션, 기프팅라인 |
| 런칭 로드맵 | 씨앗(0-3개월)→발아(3-6개월)→성장(6-12개월) 3단계 |
| 브랜드 보이스 | 상황별 톤 예시(제품소개·고객응대·SNS캡션·공지/안내) + 피해야 할 표현 |

**강점:** 가장 완성도 높은 구조. 5색 팔레트+헥스코드, 3종 타이포+스타일, 고객여정 매핑, 가격전략까지 포함
**약점:** 로고/캐릭터 설명 없음

## 1-6. Grok 보고서

**브랜드명:** 아늑한

| 섹션 | 핵심 내용 |
|------|-----------|
| 브랜드 컨셉 | "20대 감성에서 30대 진짜 나다운 공간으로 넘어가는 여성들을 위한 따뜻한 레트로 감성 셀렉샵" |
| 포지셔닝 | 단순 예쁜 소품 X → 감정과 기억을 담는 오브제. 의미 있는 소비 추구 |
| 컬러 팔레트 | Warm Beige #F5EDE4 / Terracotta #C97A5B / Moss Green #8A9A7B / Walnut Brown #5C4636 / Mustard #D9B38C (5색, 헥스코드 포함) |
| 이미지·톤 | 부드러운 자연광·필름느낌·손때 나는 질감(나무·도자기·린넨)·따뜻하고 조용하며 포근 |

# PART 2. 고객 전달 브랜딩 보고서 최종 구조 정의

> 8개 LLM 보고서 분석 + 네임텍 사업계획서 + 개발 방향을 종합하여, 네임텍이 고객에게 실제 전달할 브랜딩 보고서의 내용 구조와 각 항목의 구체적 기준을 정의한다.

---

## 2-1. 보고서 철학 및 원칙

네임텍의 결과물은 단순한 "로고 생성기"가 아닌, 창업자가 즉시 사업에 투입할 수 있는 **최소 브랜드 단위(MVB: Minimum Viable Brand)**다. 따라서 보고서는 다음 원칙을 따른다.

**원칙 1: 결과물은 항상 "왜(Why)"를 설명해야 한다**
- 컬러를 골랐으면 왜 이 컬러인지 설명
- 로고에 이 이미지를 넣었으면 왜 이 이미지인지 설명
- 캐릭터를 이 동물로 했으면 왜 이 동물인지 설명

**원칙 2: 즉시 사용 가능한 수준의 구체성**
- 컬러는 헥스코드 필수
- 폰트는 이름 + 스타일(light/regular/medium/bold) + 웨이트 수치 필수
- 카피는 예시 문장 형태로 제공
- 상황별(신상 입고·배송·고객 응대 등) 실제 카피 포함

**원칙 3: 감성과 전략의 통합**
- 단순히 예쁜 가이드가 아니라 비즈니스 관점의 포지셔닝 근거 포함

## 2-3. 섹션 A. 브랜드 아이덴티티 (Brand Identity)

> 브랜드의 존재 이유부터 실천 약속까지, 가장 핵심적인 철학적 뼈대를 담는 섹션이다. 단순한 소개를 넘어 고객과 창업자 모두가 "이 브랜드는 이런 브랜드야"라고 단번에 이해할 수 있도록 구성한다.

---

### A-1. 브랜드 철학 (Brand Philosophy)

브랜드 철학은 미션·비전보다 한 단계 위에 있는, 브랜드가 세상을 바라보는 근본적인 관점이다. "우리는 왜 이 비즈니스를 하는가?"가 아니라 "우리는 세상이 어떻게 되어야 한다고 믿는가?"에 답하는 선언이다.

**구성 항목:**

- **철학 선언문 (Brand Manifesto):** 1~2문단. 브랜드가 믿는 세계관을 감성적 언어로 표현한다. 창업자의 신념, 사회적 문제의식, 고객의 삶에 대한 통찰이 담겨야 한다.
  - 예시: "우리는 모든 사람에게 '자신만의 공간'이 필요하다고 믿습니다. 하루를 버티고 집에 돌아왔을 때, 그 공간이 나를 반겨주는 느낌. 그 작은 따뜻함이 다음 날을 버티게 하는 힘이 됩니다."

- **브랜드 존재 이유 (Raison d'être):** 1문장. 이 브랜드가 세상에 없었다면 무엇이 부족했겠는가.
  - 예시: "감각 있는 취향을 가진 사람들이 '발견'의 기쁨을 잃지 않도록."

- **브랜드가 반대하는 것 (What We Stand Against):** 3가지. 브랜드가 의도적으로 거부하는 가치나 관행. 포지셔닝을 역방향으로 명확히 한다.
  - 예시: 과소비 조장 / 일회성 유행 / 감성 없는 대량생산

- **브랜드가 지지하는 것 (What We Stand For):** 3가지. 브랜드가 적극적으로 옹호하고 실천하는 가치.
  - 예시: 오래 곁에 두고 싶은 물건 / 만든 사람의 이야기가 담긴 소품 / 일상의 작은 의식(ritual)

---

### A-2. 브랜드 에센스 (Brand Essence)

브랜드의 핵심을 단 한 단어 또는 짧은 구절로 압축한 것. 모든 의사결정의 기준이 된다.

**구성 항목:**

- **브랜드 에센스 키워드:** 1~3단어. 모든 브랜드 활동이 이 단어로 수렴해야 한다.
  - 예시: "따뜻한 발견" / "일상의 온기" / "작은 쉼표"

- **에센스 풀이:** 이 키워드가 제품, 콘텐츠, 고객 응대, 공간 등에서 어떻게 구현되는지 3~4줄로 설명.

- **브랜드 한 줄 정의 (Brand Definition):** 30자 이내. 이름과 함께 읽힐 때 즉시 이해되는 정의.
  - 예시: "퇴근 후 나를 위한 감성 소품 큐레이션"

---

### A-3. 브랜드 미션, 비전, 핵심 가치

**미션 (Mission):** 지금 현재 이 브랜드가 하는 일. 동사형으로 작성한다.
- 형식: "우리는 [누구]에게 [무엇]을 [어떻게] 제공한다."
- 예시: "우리는 바쁜 일상을 보내는 현대인에게 작은 위로와 감각적 즐거움을 주는 소품을 큐레이션하여 전달한다."

**비전 (Vision):** 5~10년 후 이 브랜드가 되고 싶은 모습. 명사형으로 작성한다.
- 형식: "[시간] 안에 [무엇]이 되는 것"
- 예시: "2030년까지 한국에서 가장 신뢰받는 감성 라이프스타일 소품 브랜드"

**핵심 가치 (Core Values):** 3~5개 키워드 + 각각 20자 이내 설명
- 형식: 키워드 / 설명 / 이 가치가 실제로 어떻게 구현되는지 예시 1줄

예시:
```
온기(Warmth)         / 차갑지 않은 것들  / 모든 소통에 계절감을 담는다
발견(Discovery)      / 예상 밖의 기쁨    / 큐레이션 기준을 고객이 알 수 없게 신선하게 유지한다
진정성(Authenticity) / 과장하지 않는다   / 만든 사람의 이야기와 소재 정보를 솔직하게 공유한다
```

---

### A-4. 브랜드명 & 네이밍 의미

- 최종 브랜드명 (국문 + 영문)
- 네이밍 선택 근거: "왜 이 이름인가?" 3~5줄 서술. 발음의 부드러움, 의미의 감성 연결, 사업 방향 연결성, 도메인/SNS 사용 가능성 포함
- 이름 탄생 스토리: 100자 이상의 서사형 설명
- **추천 도메인 및 SNS 핸들 제시** (Genspark 방식 반영: todaysondo.com / @todays_ondo 형태)
- **대안 네이밍 2~3개** + 각각의 선택 근거 (창업자가 비교 후 최종 결정 가능하도록)

---

### A-5. 슬로건

- 국문 메인 슬로건 1개 (15자 이내)
- 영문 슬로건 1개
- 슬로건 의도 설명 2~3줄
- 서브 슬로건 2~3개 (상황별 사용 가능한 변형 버전 — Genspark처럼 각 슬로건의 용도/강조점 명시)

---

### A-6. 브랜드 스토리 (Brand Story)

3~5문단 분량의 서사형 스토리. 단순 소개가 아닌, 감정 이입이 가능한 서사 구조를 따른다.

**서사 구조:**
1. **탄생 배경:** 이 브랜드가 왜 세상에 나오게 되었는가. 창업자가 느낀 불편함, 공백, 사명감.
2. **고객의 현실:** 우리 고객이 매일 마주하는 삶의 장면. 감정 이입 포인트.
3. **브랜드의 약속:** 우리가 고객에게 제공하는 것이 단순히 제품이 아닌, 어떤 경험이고 감정인지.
4. **함께하는 미래:** 이 브랜드와 함께하면 고객의 일상이 어떻게 달라지는가.

**품질 기준:** 최소 300자 이상. 브랜드 스토리 검증 시 `len(story) < 200`이면 재요청.

> **Genspark 참고:** "매일 반복되는 출퇴근, 숨 가쁘게 돌아가는 업무 시간… 우리는 완벽하게 꾸며진 화려한 공간보다, 내 취향이 온전히 묻어나는 작은 방 한편의 다정함을 믿습니다." 같이 일상의 구체적 장면에서 시작하는 서사가 효과적.

---

### A-7. 포지셔닝 선언문 (Positioning Statement)

- 형식: "For [타겟], [브랜드명]은 [경쟁군]와 달리 [차별화 가치]를 제공합니다."
- 차별화 포인트 3~5가지 (경쟁사 대비 우리가 다른 점을 항목별로 명시)
- 포지셔닝 맵: 2개 축(예: 고감성↔저감성 / 저가↔프리미엄)으로 경쟁사 대비 우리 브랜드 위치 설명

---

### A-8. 브랜드 약속 (Brand Promise)

고객에게 매번 반복적으로 전달하는 하나의 일관된 약속. 마케팅 메시지, 패키징, CS 모두에 녹아들어야 한다.

- 약속 선언 1문장: "우리는 당신에게 항상 [____]을 드립니다."
- 이 약속이 실제로 어떻게 구현되는지 채널별 적용 예시 3가지

## 2-4. 섹션 B. 타겟 고객 & 페르소나

**B-1. 핵심 타겟 정의**
- 인구통계: 연령·성별·직업·거주지역
- 라이프스타일 키워드 5개
- 소비 행태 요약

## 2-5. 섹션 C. 비주얼 아이덴티티 (Visual Identity)

**C-1. 컬러 팔레트**

필수: 5색 세트 (메인 1 + 서브 2 + 배경 1 + 텍스트 1)
각 컬러: 이름(한글) + 헥스코드 + 역할(Primary/Secondary/Accent/Neutral/Text) + 선택 근거 1줄

```
예시:
딥 어스 | #3D2B1F | Text Color | 흙과 나무껍질의 깊이감, 신뢰와 안정을 전달
샌드    | #C9A97A | Primary    | 모래사장의 따뜻함, 브랜드 핵심 온기 표현
세이지  | #7A8C6E | Secondary  | 자연 식물의 차분함, 친환경 감성 강화
리넨    | #E8DCC8 | Accent     | 린넨 원단 느낌, 촉감적 부드러움 연상
크림    | #F5F0E8 | Background | 따뜻한 여백, 모든 요소가 숨 쉬는 배경
```

**C-2. 타이포그래피**

폰트 3~4종. 각각 폰트명 + 스타일 + 웨이트 수치(숫자) + 사용 용도 + 선택 이유.

> **Genspark 방식 반영:** 한글 본문/서브 / 한글 타이틀/강조 / 영문 타이틀 / 영문 본문·숫자의 4종 분류를 기본으로 한다.

```
예시:
한글 헤드라인: 나눔명조(Noto Serif KR) / Light / 300   / 감성 강조 카피·슬로건·챕터 제목
한글 본문:    Pretendard(Noto Sans KR) / Regular / 400  / 상세페이지 본문·제품 스펙·UI 요소
영문 타이틀:  Cormorant Garamond       / Regular / 400  / 영문 로고타입·브랜드 영문명·포인트
영문 본문:    Inter                    / Regular / 400  / 가격 표기·날짜·영문 정보 전달
```

**C-3. 비주얼 무드 가이드**
- 촬영 환경 Do 3가지 (구체적 장면 묘사)
- 촬영 환경 Don't 3가지
- 무드 키워드 5개

> **Genspark 참고:** "자연광 또는 3000K 이하 전구색 조명. 여백 중심 미니멀 구도. 화이트 린넨·원목 배경. 자연스러운 그림자 유지." 처럼 촬영 현장에서 즉시 적용 가능한 수준의 구체성 필요.

**C-4. 디자인 요소 원칙**
- 레이아웃 방향 (여백, 모서리, 그림자)
- 이미지 처리 방향 (채도, 필터, 텍스처)

## 2-8. 섹션 F. 브랜드 보이스 & 톤앤매너 (Brand Voice)

**F-1. 브랜드 성격 정의**
- 5~6개 형용사로 브랜드 성격 요약
- 브랜드를 사람으로 표현하면 어떤 사람인가 (1~2줄)

> **Genspark 방식 반영:** "다정한·사려깊은·잔잔한·진솔한·섬세한·친근한" 처럼 명확한 형용사 키워드를 먼저 제시하고 설명을 붙이는 구조.

**F-2. 톤 스펙트럼 (6개 축)**

| 축 | 방향 | 비율 |
|----|------|------|
| Formal ↔ Friendly | Friendly | 70% |
| Careful ↔ Spontaneous | Careful | 65% |
| Traditional ↔ Progressive | Progressive | 55% |
| Exclusive ↔ Accessible | Accessible | 75% |
| Serious ↔ Playful | 중간-감성적 | 60% |
| Mainstream ↔ Edgy | Mainstream | 60% |

**F-3. 상황별 표현 예시 (Good/Bad 대비 — 5상황)**

> **Genspark 방식 반영:** 신상 입고 / 배송 안내 / 주말 인사 / 제품 설명 / 고객 리뷰 감사의 5가지 실제 비즈니스 상황별 카피를 포함한다.

| 상황 | 권장 표현 | 금지 표현 |
|------|-----------|-----------|
| 제품 소개 | "창가에 두면 오후 햇살을 닮아가는 도자기 머그입니다." | "인기폭발! 지금 바로 담으세요!" |
| 고객 응대 | "소중한 주문 감사합니다. 따뜻하게 포장하여 보내드릴게요." | "네 처리해드리겠습니다." |
| SNS 캡션 | "퇴근 후, 차 한 잔과 함께하는 시간 ☕ 오늘은 어떤 하루였나요?" | "NEW ARRIVAL 🔥 지금 구매시 할인!" |
| 공지/안내 | "잠시 숨 고르는 시간을 갖고, 더 좋은 모습으로 찾아올게요." | "공지: 시스템 점검 실시합니다." |
| 배송 안내 | "기다리시는 마음을 알기에, 작은 엽서 한 장에 온기를 담아 꼼꼼히 포장했습니다." | "상품 발송 완료. 운송장 번호 확인바랍니다." |

**F-4. 절대 사용 금지 표현 목록**
- 과도한 세일 언어 (혜자, 가성비, 핫딜, 득템)
- 과장형 감탄사 (대박, 미쳤다, 레전드)
- 조급함 유발 언어 (지금 당장, 품절 임박, 마지막 기회)
- 수치 강조형 (99% 만족, 1위, 최고)
- 가성비 강조 (저렴하게, 싸게, 가격 대비)

## 2-9. 섹션 G. 콘텐츠 전략 (Content Strategy)

> 네임텍이 생성하는 콘텐츠 전략은 단순 "포스팅 방향" 수준을 넘어, 브랜드가 어떤 철학으로 콘텐츠를 만들고 어떤 채널에서 어떻게 운영하며 어떤 결과를 목표로 하는지를 실행 가능한 수준으로 제시한다.

---

### G-1. 콘텐츠 철학 (Content Philosophy)

콘텐츠는 광고가 아니라 브랜드의 세계관을 전달하는 창구다. 이 브랜드의 콘텐츠는 다음 원칙을 따른다.

- **첫 번째 원칙: 팔기 전에 머물게 한다.** 구매를 유도하기 전에 고객이 이 브랜드의 콘텐츠 안에서 자발적으로 시간을 보내고 싶도록 만든다.
- **두 번째 원칙: 제품이 아닌 장면을 판다.** 머그컵이 아니라 그 머그컵을 들고 앉아 있는 오후의 장면을 판다. 소품이 아니라 그 소품이 있는 공간의 감정을 판다.
- **세 번째 원칙: 일관성이 신뢰를 만든다.** 한 번의 멋진 게시물보다 매주 쌓이는 일관된 세계관이 팬을 만든다. 모든 콘텐츠는 같은 감성 레이어 안에 있어야 한다.

---

### G-2. 3대 콘텐츠 기둥 (Content Pillars)

브랜드의 모든 콘텐츠는 3개의 기둥 중 하나에 속한다. 비율은 기둥1: 50%, 기둥2: 30%, 기둥3: 20%를 권장한다.

**기둥 1. 감성형 콘텐츠 (Emotional Content)** — 비율 50%

- **목적:** 브랜드 세계관을 전달하고 감정적 연결을 형성한다. 팔로우와 저장을 유도한다.
- **주요 형식:**
  - 공간 연출 사진: 제품이 놓인 일상의 장면. 인물 없이 분위기만으로 이야기하는 사진.
  - 계절·날씨 연동 콘텐츠: "이런 날씨엔 이런 소품이", "오늘 같은 흐린 월요일에"
  - 감성 텍스트 카드: 브랜드 철학을 담은 짧은 문장을 카드 형태로 시각화
  - 루틴 장면: 퇴근 후, 아침 커피, 주말 오전 등 타겟 고객의 일상 의식(ritual)을 담은 콘텐츠
- **발행 주기:** 인스타그램 피드 주 2~3회, 릴스 주 1회
- **KPI:** 저장수, 팔로워 증가, 도달 수

**기둥 2. 발견형 콘텐츠 (Discovery Content)** — 비율 30%

- **목적:** 제품의 가치와 큐레이션 기준을 설명하고 구매 욕구를 자연스럽게 자극한다.
- **주요 형식:**
  - 셀렉터 노트: "이번 달 우리가 고른 이유" — 신상품의 큐레이션 기준을 설명하는 글
  - 소품 스토리: 이 제품이 어디서 왔는지, 누가 만들었는지, 어떤 소재인지 설명
  - 비하인드 큐레이션: 수백 개 중에서 이 소품을 선택한 과정
  - 사용 방법 제안: "이렇게도 쓸 수 있어요" — 제품의 새로운 활용법
  - 비교 콘텐츠: 비슷해 보이는 두 소품의 차이를 설명 (소재, 질감, 분위기)
- **발행 주기:** 인스타그램 피드 주 1~2회, 스마트스토어 상세페이지 업데이트
- **KPI:** 상세페이지 클릭률, 장바구니 담기, 댓글 수

**기둥 3. 관계형 콘텐츠 (Relationship Content)** — 비율 20%

- **목적:** 고객과의 양방향 소통으로 커뮤니티를 형성하고 재구매를 유도한다.
- **주요 형식:**
  - 고객 공간 소개: "우리 소품이 살고 있는 공간들" — 구매 고객의 실사용 공간 공유
  - 소품 투표: "A vs B, 어떤 게 더 당신의 공간에 어울리나요?"
  - 스토리 Q&A: 소품 큐레이션 기준, 브랜드 운영 이야기에 대한 질문 받기
  - 계절 맞이 기획: 봄/여름/가을/겨울 감성 소품 기획전 사전 예고 및 참여 유도
  - 뉴스레터 전용 콘텐츠: 구독자 전용 에세이, 먼저 보는 신상품
- **발행 주기:** 인스타그램 스토리 주 3~4회, 뉴스레터 월 2회
- **KPI:** 스토리 응답률, 뉴스레터 오픈율, 재구매율

### G-3. 채널별 운영 전략

**채널 1. 인스타그램 (Instagram)**

- **역할:** 브랜드 세계관의 쇼룸. 첫 발견과 팔로우 전환의 핵심 채널.
- **피드 구성 원칙:**
  - **9그리드 패턴** 활용: [제품 풀샷 - 일상 에세이/풍경 - 제품 디테일]을 반복 배치하여 피드 전체가 하나의 잡지처럼 보이도록 구성 (Genspark 반영)
  - 가로 3열 기준으로 볼 때 색감과 분위기가 일관되어야 한다
  - 제품 단독 컷보다 공간 연출 컷을 7:3 비율로 유지
  - 텍스트가 과하게 들어간 이미지 지양
  - 무채색 배경보다 따뜻한 톤의 자연 소재 배경 권장
- **릴스 전략:**
  - 형식: 15~30초 짧은 분위기 영상 (ASMR형, 슬로우모션, 공간 전환 연출)
  - 주제 예시: "퇴근 후 15분 디스크레스팅", "이 소품 하나로 달라진 책상", "포장 ASMR (바스락거리는 종이 소리)" (Genspark 반영)
  - 첫 3초 안에 감성적 장면 또는 호기심 유발 문장 삽입
- **콘텐츠 주제 예시 10선** (Genspark 반영):
  1. (제품) "비 오는 주말, 방안을 채우는 우드향" — 인센스 홀더 연출컷
  2. (일상) 큐레이터의 퇴근 후 1시간 플레이리스트 공유
  3. (디테일) 컵 표면의 미세한 도자기 결을 확대한 접사 사진
  4. (스타일링) 좁은 책상 위 빈티지 문구들로 만드는 데스크테리어 팁
  5. (고객후기) 고객이 남겨준 사진과 다정한 리뷰 리포스트
  6. (신상) "이번 달 우리의 시선이 머문 물건들" — 신상 스포일러
  7. (에세이) 오래된 물건이 주는 위로에 대한 짧은 글귀
  8. (패키징) 포장 과정 릴스 (ASMR)
  9. (이벤트) "당신의 방에 가장 잘 어울릴 색깔 고르기" 댓글 이벤트
  10. (제품) 계절의 변화를 느낄 수 있는 패브릭 포스터 활용법
- **해시태그 전략:** (Genspark 반영)
  - 게시물 성격에 맞춰 5~8개를 조합하여 사용
  - 권장 해시태그: #감성소품 #일상기록 #홈스타일링 #방꾸미기 #데스크테리어 #빈티지소품 #나만의공간 #온라인셀렉샵 #취향공유 #소확행 #인테리어소품
- **발행 주기:** 피드 주 3~4회, 릴스 주 1회, 스토리 주 4~5회
- **성장 전략:**
  - 초기(0~3개월): 15개 이상 핵심 게시물 미리 준비 후 런칭 (Genspark 방식)
  - 중기(3~6개월): 마이크로 인플루언서(팔로워 5k~20k, 집꾸미기/다꾸 계정) 5명 협업
  - 후기(6개월+): 고객 UGC 공식 채널 공유, "이달의 다정한 방" 선정 이벤트

**채널 2. 네이버 스마트스토어**

- **역할:** 실제 구매가 일어나는 핵심 커머스 채널.
- **상세페이지 구성 원칙:**
  - 오프닝: 감성적 장면 사진 + 1~2줄 감성 카피 (제품명 바로 노출 X)
  - 스토리: 이 소품을 왜 선택했는지, 어떤 상황에 어울리는지
  - 스펙: 소재, 크기, 생산지, 관리 방법 (간결하게, 표 형태)
  - 리뷰 하이라이트: 실사용 고객의 공간 사진 + 한 줄 후기
  - 마무리: 패키징 소개 + 배송 안내 (감성적 언어로)
- **상품명 작성 규칙:** 감성 키워드 + 소재/특징 + 제품명 조합

**채널 3. 뉴스레터 (Newsletter)**

- **역할:** 가장 충성도 높은 고객과의 1:1 대화 채널. 재구매와 팬화의 핵심.
- **발행 주기:** 격주 1회 (월요일 오전 7시 권장)
- **구성 형식:** 오프닝 에세이 + 이번 호의 소품 + 뉴스레터 전용 혜택 + 마무리 인사

**채널 4. 블로그 / 브런치**

- **역할:** 검색 유입 + 브랜드 스토리의 롱폼 아카이브.
- **발행 주기:** 월 2~4회

## 2-10. 섹션 H. 상품 큐레이션 가이드 (Product Curation Guide)

> **신규 섹션 — Genspark 보고서에서 도출.** 창업자가 실제 상품을 선정하고 구성하는 데 즉시 활용할 수 있는 기준과 프레임워크를 제공한다.

---

### H-1. 큐레이션 철학 & 3원칙

모든 상품은 다음 3가지 원칙을 충족해야 한다. 원칙을 통과하지 못한 상품은 입점 보류한다.

1. **일상에 자연스레 스며들 것:** 너무 화려하거나 튀지 않아 매일 보아도 질리지 않는 디자인.
2. **손맛이 느껴질 것:** 기성품의 차가움보다는 핸드메이드나 빈티지가 주는 고유의 결이 있는 제품.
3. **오래 두고 쓸 것:** 일회성이 아닌, 시간이 지날수록 사용자의 손때가 묻어 멋스러워지는 소재(우드, 린넨, 도자기, 황동 등).

**추가 판단 기준:**
- 이 상품은 "예쁜가?"를 넘어 "오래 두고 싶은가?"에 답할 수 있는가
- 소재와 만든 사람에 대한 이야기를 쓸 수 있는가
- 공간에 배치했을 때 "장면"이 만들어지는가

### H-2. 추천 상품 카테고리 (6종)

| 카테고리 | 설명 | 대표 아이템 예시 |
|----------|------|-----------------|
| 빈티지 문구 & 페이퍼 | 낡은 서랍 속에서 발견한 듯한 감성 문구류. 데스크테리어 특화. | 영문 페이퍼, 빛바랜 메모지, 황동 클립, 감성 스탬프 |
| 핸드메이드 캔들 & 디퓨저 | 인공적이지 않은 자연의 향. 오브제로서의 역할도 충족. | 소이 캔들, 인센스 홀더, 비정형 왁스 타블렛 |
| 드라이플라워 & 식물 소품 | 시들지 않는 다정함. 방 안에 자연을 들이는 소품. | 드라이플라워 엽서, 빈티지 화병, 테라리움 |
| 빈티지 식기 & 컵 | 혼자만의 티타임을 특별하게 만드는 오브제. | 도자기 머그, 패턴 유리컵, 우드 티코스터 |
| 패브릭 & 홈 텍스타일 | 공간의 분위기를 부드럽게 전환하는 천 소재 제품. | 리넨 가림막, 패브릭 포스터, 테이블 매트 |
| 손글씨 카드 & 다이어리 | 종이 질감과 쓰는 경험을 중시하는 문구. | 서걱거리는 다이어리, 만년필 노트, 메시지 카드 |

### H-4. 패키징 가이드 (체크리스트)

> **Genspark 보고서의 핵심 강점 — 체크리스트 형태로 즉시 실행 가능하게 제공**

언박싱 순간부터 브랜드의 다정함이 느껴지도록 포장재에도 브랜드 감성을 일관 적용한다.

**기본 패키징 구성:**
- [ ] **외관:** 환경을 생각하는 무지 크라프트 박스 사용
- [ ] **완충재:** 비닐 뽁뽁이 대신 친환경 종이 완충재(지아미) 활용
- [ ] **동봉물:** 고객의 이름을 손글씨(또는 손글씨 폰트 인쇄)로 적은 감사 엽서
- [ ] **포인트:** 드라이플라워 한 줄기 또는 브랜드 컬러의 왁스 실링 스탬프로 마감

**패키징 Do & Don't:**

| DO | DON'T |
|----|-------|
| 종이 테이프, 무지 박스 등 친환경·감성 소재 사용 | 화려한 로고가 박힌 비닐 택배 봉투 |
| 크라프트 박스 + 지아미 + 손글씨 엽서 세트 구성 | 과도한 플라스틱 뽁뽁이 |
| 드라이플라워·왁스 실링 등 작은 서프라이즈 요소 | 브랜드 무드와 맞지 않는 쨍한 색상 리본 |

## 2-11. 섹션 I. 브랜드 실행 가이드 (Execution Guide)

**I-1. 브랜드 오픈 전 체크리스트** (Genspark 12항목 반영)

- [ ] 브랜드명 상표출원 (등록 가능 여부 사전 확인)
- [ ] 도메인 및 주요 SNS 계정(인스타그램) 선점 완료
- [ ] 메인/서브 컬러 HEX 코드 및 웹폰트 사이트 내 적용 완료
- [ ] 로고 파일 (원본 AI, 투명 PNG — 웹용·인쇄용) 정리 완료
- [ ] 포장 패키징 부자재 (크라프트 박스, 지아미, 스탬프, 엽서) 발주 완료
- [ ] 첫 판매를 위한 메인 큐레이션 상품 (5~7종) 사입 및 샘플 확인 완료
- [ ] 자연광 스튜디오 또는 공간 대관하여 1차 상품 누끼/감성컷 촬영 완료
- [ ] 상세페이지 템플릿(브랜드 스토리 + 제품 설명 + 배송/교환 안내) 제작 완료
- [ ] 배송 동봉용 감사 엽서 디자인 및 발주 완료
- [ ] 인스타그램 초기 세팅용 콘텐츠(이미지+캡션) 9~15개 분량 사전 제작 완료
- [ ] CS 가이드라인(배송 지연, 환불 등)에 브랜드 보이스 반영 완료
- [ ] 결제 시스템(PG) 및 택배사 연동 테스트 1회 이상 완료

**I-2. 런칭 단계별 로드맵 (Phase 3단계)**

| Phase | 기간 | 목표 | 핵심 액션 |
|-------|------|------|-----------|
| Phase 1 씨앗 | 0~3개월 | 브랜드 세팅 완료 + 첫 컬렉션 출시 | 콘텐츠 30개 사전 제작, 핵심 소품 10종 셀렉, 인스타 1일 1포스팅 |
| Phase 2 발아 | 3~6개월 | 팔로워 1K + 첫 단골 50명 확보 | 마이크로 인플루언서 3인 협업, 뉴스레터 런칭, 시즌 기획전 1회 |
| Phase 3 성장 | 6~12개월 | 월 매출 안정화 + 재구매율 40%+ | 팝업스토어 1회, 고객 커뮤니티 운영, PB(자체브랜드) 상품 개발 |

**I-3. 핵심 KPI (6개월 목표)**

| 지표 | 목표 |
|------|------|
| 인스타그램 팔로워 | 3,000명 |
| 뉴스레터 구독자 | 500명 |
| 자사몰 전환율 | 4% |
| 재구매율 | 60% |

## 2-12. 패키지별 포함 범위

| 패키지 | 포함 섹션 | 설명 |
|--------|-----------|------|
| A (기초 브랜드) | A + B(일부) + F(일부) | 브랜드명·슬로건·스토리·기본 톤앤매너 |
| B (브랜드 키트) | A + B + C + F | + 컬러·타이포·비주얼가이드·보이스 전체 |
| B+ (로고 포함) | A + B + C + D + F | + 로고 아이덴티티 |
| B++ (캐릭터 포함) | A + B + C + D + E + F | + 캐릭터 가이드 |
| C (디지털) | A + B + C + D + F + G | + 콘텐츠 전략·카피 예시 |
| D (풀 패키지) | A~I 전체 | + 상품 큐레이션 가이드·실행 가이드·로드맵·KPI |

# PART 7. PDF 생성 설계

WeasyPrint + Jinja2 조합을 사용한다.

## 7-1. PDF 생성 방식: 패키지별 점진적 빌드(Layered Build)

> 고객이 구매한 패키지 수준에 따라 PDF를 점진적으로 확장 생성한다. "A만들고 → B 덧붙이고 → C 덧붙이고"의 레이어 방식으로 구현한다.

```
[패키지 A — 기초 브랜드]
  표지 + 브랜드 철학 + 에센스 + 미션/비전/가치 + 브랜드 스토리
  + 네이밍/슬로건 + 기본 톤앤매너

       ↓ 패키지 B 추가
[패키지 B — 브랜드 키트]
  위 내용 + 컬러 팔레트 + 타이포그래피 + 비주얼 무드 가이드
  + 브랜드 보이스 전체 (상황별 카피 예시 포함)

       ↓ 패키지 B+ 추가
[패키지 B+ — 로고 포함]
  위 내용 + 로고 아이덴티티 (로고 이미지 + 컨셉 설명)

       ↓ 패키지 B++ 추가
[패키지 B++ — 캐릭터 포함]
  위 내용 + 캐릭터 가이드 (이름 + 선택근거 + 스토리)

       ↓ 패키지 C 추가
[패키지 C — 디지털]
  위 내용 + 콘텐츠 전략 (3대 기둥 + 채널 전략 + 인스타 9그리드)
  + 카피 예시 패키지 + 해시태그 전략

       ↓ 패키지 D 추가
[패키지 D — 풀 패키지]
  위 내용 + 상품 큐레이션 가이드 (원칙 + 카테고리 6종 + 패키징 체크리스트)
  + 브랜드 실행 가이드 (오픈 전 체크리스트 + 런칭 로드맵 + KPI)
```

**구현 방식:** Jinja2 템플릿에서 `{% if package_type in ['B', 'C', 'D'] %}` 조건 블록으로 섹션 포함 여부를 제어. WeasyPrint가 최종 HTML을 렌더링하여 단일 PDF 파일로 출력.

## 7-3. PDF 디자인 원칙

- **폰트:** 헤드라인 = 명조계열(Noto Serif KR), 본문 = 고딕(Pretendard/Noto Sans KR)
- **컬러:** AI가 생성한 브랜드 컬러를 실제 PDF 배경/액센트에 동적 적용
- **여백:** 섹션 간 충분한 여백으로 고급스러운 인상
- **컬러 스와치:** 실제 HEX 컬러가 렌더링되는 사각형 블록
- **타이포그래피 샘플:** 각 폰트를 해당 폰트로 실제 렌더링
- **페이지 번호:** 하단 중앙, 브랜드 컬러 점(·)으로 구분
- **체크리스트:** 실제 체크박스 UI 형태로 렌더링 (Genspark 방식 반영)

## 9-1. 에러 처리 정책

| 상황 | 처리 방식 |
|---|---|
| 세션 없음 | `404 Not Found` + "세션을 찾을 수 없습니다." |
| 잘못된 Phase 순서 | `400 Bad Request` + "이전 단계를 먼저 완료하세요." |
| AI JSON 파싱 실패 | 최대 2회 재시도 후 `500` 반환 |
| 잘못된 후보 id 선택 | `422 Unprocessable Entity` |
| PDF 생성 실패 | `500` + 로그 기록 |
| 이미지 생성 실패 | 3회 재시도 → 실패 시 텍스트만으로 PDF 완성 |

---

## 9-2. 기술적 주의사항

**1. AI 프롬프트에서 JSON만 출력하도록 강제**
- 시스템 인스트럭션에 "JSON만 출력, 마크다운 코드블록 금지" 명시
- `parser.py`에서 ```json ``` 블록 자동 제거 로직 유지

**2. Phase 4 프롬프트는 선택된 내용을 정확히 삽입**
- Phase 3에서 저장된 `selected.brand_name`, `selected.slogan` 등을 프롬프트에 하드코딩
- AI가 임의로 브랜드명을 바꾸지 못하게 함

**3. PDF의 컬러는 AI 생성값을 그대로 사용**
- `brand_guide_template.html`에서 Jinja2 변수(`{{ color_palette[0].hex }}`)로 CSS 변수에 직접 바인딩

**4. 브랜드 스토리는 300자 이상 강제**
- 프롬프트에 "300자 이상" 명시
- 응답 검증 시 `len(story) < 200`이면 재요청

**5. 타이포그래피는 폰트 웨이트(숫자)까지 명시 강제**
- 프롬프트에 예시(`"weight": 400`)를 포함시켜 숫자형으로 출력 유도
- 4종 분류(한글 헤드라인/한글 본문/영문 타이틀/영문 본문) 준수 확인

**6. 세션 상태 일관성 보장**
- session_id를 localStorage 또는 쿠키로 클라이언트에 보관
- 서버에서 복원 가능하도록 설계

**7. PDF 한글 폰트**
- WeasyPrint에서 한글 폰트 깨짐 방지를 위해 서버에 Noto Sans KR 폰트 미리 설치
- CSS `@font-face`로 명시적으로 지정

**8. 패키지별 레이어 빌드 검증**
- PDF 생성 전 `package_type` 값 검증
- 각 패키지가 상위 패키지의 모든 섹션을 포함하는지 확인
- A ⊂ B ⊂ B+ ⊂ B++ ⊂ C ⊂ D 관계 보장

**9. 도메인·SNS핸들 제안 검증**
- AI가 제안한 도메인이 실제 가용한지는 검증 불가능하므로, PDF에 "실제 가용 여부는 직접 확인 필요" 안내 문구 삽입

**10. 비용 관리**
- 후보 3개 병렬 생성은 토큰 비용이 높으므로 max_tokens를 1,500으로 제한
- 최종 가이드 생성 단계에서 섹션별로 분할 호출 검토

**11. 프롬프트 인젝션 방지**
- 사용자 입력값을 프롬프트에 삽입할 때 반드시 이스케이프 처리
- 사용자 입력과 AI 지시 영역을 구조적으로 분리
